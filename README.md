# [bmp](./bmp/bmp.md)  
bmp和yuv420 plat格式的图片文件相互转换  
运行环境：linux，编译器：gcc  
文件说明：  
	bmp.h			----- 声明bmp头结构， bmp2yuv.c和yuv2bmp.c公用  
	yuv2bmp.c	----- yuv图片转换为bmp格式程序  
	bmp2yuv.c	----- bmp图片格式转为yuv格式程序
